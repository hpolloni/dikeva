Requirements
-----------
Python 2.5

Installation
------------
To install, decompress this on a directory where it could be read by a python interpreter

Usage
-----
To start a server, you'll need the following command:

$> python (PATH_TO_DIKEVA)/run_server.py --config=config.cfg


Python Client API
-----------------
 checkout the test1-put.py and test2-get.py files. That will give you a general idea to how to use the DkvClient class.
